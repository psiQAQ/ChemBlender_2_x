## Known Limits

- The pinned `cp313-win_amd64` RDKit wheel limits 2.2.0 to Blender 5.1+ on Windows x64.
- Overwriting an already loaded local installation may emit Windows DLL file-lock cleanup warnings; the lifecycle test still passed. Release CI uses a clean runner.
- `mesh.py` and `read.py` retain pre-existing invalid-regex-escape warnings.
- GitHub Actions has not run until the branch is pushed.
