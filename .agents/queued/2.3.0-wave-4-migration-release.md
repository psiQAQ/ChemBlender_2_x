# ChemBlender 2.3.0 Wave 4 — Migration and Release Qualification

## State

Queued behind Wave 3.

## Goal

交付 `2.3.0-rc.1`，完成旧路径统一、迁移、CI、性能、文档和最终2.3.0发布门。

## Success Criteria

- 旧File/SMILES/PubChem/CIF/POSCAR UI调用统一Import Pipeline。
- Legacy `.blend`显式迁移、备份和rollback。
- CI分层，目标真实integration不得skip。
- 动态版本/artifact、prerelease/final release正确。
- artifact size、wheel inventory、licenses和性能门通过。
- 用户、插件、迁移、故障排查和格式能力文档完整。
- RC后无新范围。

## Constraints

- 不删除仍有调用者或未迁移fixture覆盖的legacy代码。
- Release只使用exact-tag成功artifact，不重建。

## Verification

完整CPython、optional integration、Blender build/install/lifecycle、migration fixtures、performance、artifact audit和read-only release dry-run。
