# MOL V2000 Reader Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 用 Python 标准库读取单记录 MOL V2000，并证明其 normalized structure 与现有 XYZ fixture 一致。

**Architecture:** 新增一个与 `xyz.py` 同级的 reader 模块，复用现有 `Structure`、`ImportBatch`、reader descriptor 和元素表。只读取结构；键和属性通过 `ParserReport` 显式标记，不增加 `Topology`。

**Tech Stack:** Python 标准库 `array`、`hashlib`、`math`、`pathlib`、`uuid`、`unittest`。

## Constraints

- 不新增依赖、模型 registry、Blender UI 或旧 RDKit 路径修改。
- 只支持单记录 MOL V2000；V3000 与 SDF 明确拒绝。
- 非法或截断输入不返回部分 batch。
