# 量子化学可视化开发

ChemBlender 将继续作为 Blender 原生可视化前端；量子化学解析、语义归一化和大型数组管理放在不依赖 `bpy` 的数据边界之后。开发按物理量与数据责任组织，不按文件后缀逐项增加按钮。

## 当前阶段

Phase 0、Phase 1 分子闭环、Phase 2 周期量子化学与 Phase 3 大型数据基础设施均已完成。Phase 4 recipe、external process、critic2 topology、QCSchema/CJSON exchange、QCEngine/PySCF execution、analysis report 与 scene preset contract 已验收，当前实施 preset Blender application；第三方 parser、数值后端与外部分析继续保留在独立 core/worker 环境。
